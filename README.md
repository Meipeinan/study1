学习中得到的：
1.重置样式的目的
使样式在不同浏览器下保持一致
2.块级元素的高度是默认的，文字无法撑起高度，默认就为0
3.height:inherit（继承父元素的高度）
4.css3中属性：transform:rotate(45deg);用于旋转45度。
在实现过程中发现搜狗浏览器不支持css3样式，不能显示出旋转的效果。
解决办法：-webkit-transform:rotate(45deg);
不同浏览器有不同的内核，对于一些CSS3特效是有不同书写要求的。
